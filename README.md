## 基于Slingshot的React基础项目

包括React-redux、React-thunk、Eslint、Jest、SASS等

开始：

```
yarn (or :npm install)
yarn start (or: npm start)
//http://localhost:3000/
//简单实现redux counter和thunk异步加载
```
